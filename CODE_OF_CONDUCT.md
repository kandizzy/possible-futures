# Code of Conduct

This project exists to help people navigate a brutal job market with more agency than ATS systems allow them. Discussions here should reflect that humane intent.

## Be kind

If you're here to report a bug, suggest a feature, submit a patch, or ask a question — you're welcome. Bring the same patience and good faith you'd want extended to you on a bad day of job searching.

## What's not okay

- Harassment, sustained hostility, or personal attacks. This includes demeaning comments based on gender, sexuality, race, religion, disability, nationality, body, age, or any other identity axis.
- Sharing someone else's personal or professional information without consent.
- Spam, advertising, or promotion of unrelated products or services.

## Scope

This applies in the GitHub repository — issues, pull requests, discussions, commit messages, and review comments. It also applies to any communication identified as being about this project, regardless of venue.

## Enforcement

To report a concern privately, open a [GitHub security advisory](https://github.com/kandizzy/possible-futures/security/advisories/new). It's the only private communication channel GitHub offers small repos, so it serves double duty for code-of-conduct reports as well as security. For less sensitive matters, you can @mention [@kandizzy](https://github.com/kandizzy) in an issue or discussion.

I read every report. I'll respond within a week, preferably sooner. Consequences for breaking the rules range from a private note, to a public warning, to a temporary block, to a permanent ban from the repository — proportional to severity and any repeat pattern.

## Attribution

This code of conduct takes cues from the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) but is deliberately shorter and written in the voice of this project.
